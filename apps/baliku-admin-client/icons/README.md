# Icons untuk PWA Admin

Folder ini berisi icons untuk Progressive Web App Admin.

## Required Icons

Untuk compile ke APK, Anda perlu 2 icon sizes:

1. **icon-192.png** - 192x192 pixels
2. **icon-512.png** - 512x512 pixels

## Cara Generate Icons

### Opsi 1: Online Generator (Paling Mudah)

1. Kunjungi: https://www.pwabuilder.com/imageGenerator
2. Upload logo (gunakan logo yang sama atau buat variasi untuk admin)
3. Download generated icons
4. Extract ke folder ini

### Opsi 2: Manual dengan Image Editor

1. Buka logo di image editor (Photoshop, GIMP, dll)
2. Resize ke 192x192 pixels
3. Save sebagai `icon-192.png`
4. Resize ke 512x512 pixels
5. Save sebagai `icon-512.png`

### Opsi 3: Online Tools

- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
- https://favicon.io/

## Catatan

- Format: PNG dengan transparansi
- Background: Transparan atau solid color (#0b1020)
- Quality: High quality untuk 512x512
- Purpose: "any maskable" untuk Android adaptive icons
